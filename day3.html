Difference Between Let Var and Const Variable

Const Var Let
The const keyword has all the properties that are the same as the let keyword, except the user cannot update it and have to assign it with a value at the time of declaration. The var is the oldest keyword to declare a variable in JavaScript. The let keyword is an improved version of the var keyword.
